`Color`
--

`> 单词`

`> RGB`

`> 十六进制表示法` `#000 #333 #999 #fff #ff6767`

`> RGBA`

`> HSLA 表示法：background-color: hsla(240,50%,50%,0.4)`

`! H 色调，取值范围 0~360 0 或 360 表示红色 120 表示绿色 240 表示蓝色`

`! S 饱和度 取值范围 0%~100% 值越大越鲜艳`

`! L 亮度 取值范围 0%~100% 亮度最大时为白色最小时为黑色`

`! A 透明度 取值范围 0~1`
